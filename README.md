## 协作流程

1. 组长创建原始仓库 `innovation-week15-team-device-9`
2. 组员 Fork 仓库并创建个人分支
3. 每名组员只修改自己负责的区域
4. 组员提交 Pull Request
5. 组长 Review 并合并
6. 主电脑连接 Android 手机运行最终版本
7. 第二部手机拍摄手持真机照片
8. README 展示分工、PR 和照片

## 运行说明

# 获取依赖
flutter pub get

# 运行到 Android 真机
